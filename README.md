DevOps CI/CD – Node.js, Docker & AWS
Projet démontrant la mise en place d’un pipeline CI/CD complet pour déployer automatiquement une application Node.js sur une instance AWS EC2.
Fonctionnalités :
- Conteneurisation de l’application avec Docker
- Pipeline GitHub Actions (build → test → déploiement)
- Connexion SSH automatisée vers AWS
- Déploiement continu à chaque push
Objectifs :
- Comprendre un workflow DevOps complet
- Automatiser les déploiements
- Gérer une infrastructure cloud
